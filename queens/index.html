Têmpera Simulada (Simulated Annealing)
A cada passo, escolhe um vizinho aleatório. Aceita se for melhor; se for pior, aceita com probabilidade e−ΔE/T. A temperatura T diminui gradualmente (T *= α), convergindo para a solução.
